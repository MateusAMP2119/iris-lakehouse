# E05.7 Dead letter replay

Epic: [[Iris Epics]], E05 Dispatcher, Lanes and Dead Letters
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Deliver the dead-letter worklist and replay. dead_letters holds one row per outstanding dead-lettered run awaiting disposition, and an entry leaves only via replay, supersession, or drain while the run row stays in runs. Replay mints a fresh run through the normal lane path with cause replay and replayed_from chaining, walks propagated entries to their root cause, never force-runs dependents, and propagated entries self-supersede once their dependent consumes a later upstream run.

## Depends on

Epics E02, E03; builds on [[E05.6 Failure propagation]].
