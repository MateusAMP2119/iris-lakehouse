# E14.1 Ref grammar and triage shows

Epic: [[Iris Epics]], E14 Graph Views and Triage Surface
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

The run ref grammar and the triage folded into the shows. Everywhere a command accepts <run>, a bare pipeline name resolves to its latest run and <name>~n to the nth prior run. The gate ledger in pipeline show re-exposes the dispatcher's eligibility query as code reuse, the blast radius in deadletter show classifies transitive downstream, and run show --trace walks run_inputs for ancestry. These read surfaces mint no meta state and reuse existing engine walks rather than adding new logic.
