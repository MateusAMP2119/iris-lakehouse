# E00.4 Fakes and process IO

Epic: [[Iris Epics]], E00 Conformance Harness and Traceability Gate
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Put store, pg, and exec behind interfaces satisfied by fakes (recording pg fake capturing DDL as goldens, meta-store fake, fake process runner for dispatch tests), so integration tests run with no real process or database. Cover real process I/O that needs no database (subprocess execution, output capture, cancel/kill, socket HTTP against the in-process daemon) against throwaway scripts, and adopt the no-fixed-sleeps convention for async assertions.

## Depends on

[[E00.3 Golden files and fixtures]] (the recording pg fake writes golden files).

| Contract | Behavior | Tier |
