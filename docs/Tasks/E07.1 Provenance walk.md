# E07.1 ==Provenance walk==

Epic: [[Iris Epics]], E07 Provenance, Journal Lifecycle and Object Store
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Implement the ==`iris data provenance`== three-lookup walk as pure query logic: row to stamps, run id to run facts, and recursive ancestry via run_inputs. The walk returns lineage only, never row images, and falls back to archival summaries so provenance keeps naming the exact declaration checksum and binary hash after run rows are pruned. This is the unit-heavy read-side foundation the rest of the epic closes over.
