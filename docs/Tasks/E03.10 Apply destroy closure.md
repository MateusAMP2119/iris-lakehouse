# E03.10 Apply destroy closure

Epic: [[Iris Epics]], E03 Declarations, Schemas and Apply
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Close the epic against the live daemon: scoped single-unit destroy that retires all of a pipeline's rows, bytes, and grants in one meta transaction, reverts un-promoted disposable data, honors the composer destroy interlock, and end-to-end conformance that apply is idempotent and strictly single-file. This is the last cluster per the cutting notes, since it exercises apply and destroy against a running engine.

## Depends on

Epics E01, E02; builds on [[E03.8 Idempotent provisioning]] and [[E03.9 Registry persistence in meta]].
