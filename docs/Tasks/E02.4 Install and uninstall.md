# E02.4 Install and uninstall

Epic: [[Iris Epics]], E02 Engine Install, Daemon and Leadership
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Wire the full iris engine install and uninstall commands over the pieces built so far: install bootstraps meta via plain CREATE DATABASE, ensures tables and the partitioned journal, mints the ed25519 engine key, and sets up the socket; iris engine info exposes only the key's public half. Uninstall is a full teardown that drops meta, journal, endpoints, object store, socket, and service unit, leaving nothing behind.

## Depends on

Epic E01; builds on [[E02.1 Meta DDL and schema]], [[E02.2 Admin DSN chain]], [[E02.3 Managed Postgres subprocess]].
