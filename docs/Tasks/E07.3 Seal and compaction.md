# E07.3 Seal and compaction

Epic: [[Iris Epics]], E07 Provenance, Journal Lifecycle and Object Store
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Implement the journal partition seal condition and the compaction pass: null released pre-images past undo eligibility and collapse duplicate stamps to the latest op while preserving each run's exact write set. The compaction collapse rule is pure unit logic; sealing then runs as an opportunistic dispatcher step (compact, checkpoint, archive) after a pass.
