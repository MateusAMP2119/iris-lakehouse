# E10.3 Remote tiering and failover

Epic: [[Iris Epics]], E10 Destructive Operation Gates
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Close the remote-surface tiering and live-daemon guarantees of the gates: which destructive ops are reachable over the TCP listener, API invocation requiring a control PAT plus explicit confirm field and leader-only acceptance, uninstall staying strictly local (no listener path, refusing while a daemon candidate holds a meta connection), and a new leader after failover never resuming an interrupted destructive op. This is the live-daemon conformance closure of the epic.

## Depends on

Epics E03, E05, E06. Builds on [[E10.2 Confirmation flows]].
