# E13.2 Install and binary boot

Epic: [[Iris Epics]], E13 Golden Sample and Acceptance
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Prove the definition-of-done checks around getting an engine on the ground: install creates the meta database alongside the data database, install plus start covers both managed local Postgres and external mode through one code path, and the engine cross-compiles to a single static binary that boots with no host runtime. The cross-compile smoke run exercises boot, install, start, apply, and teardown across linux and macos on amd64 and arm64.
