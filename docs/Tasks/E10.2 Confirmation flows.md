# E10.2 Confirmation flows

Epic: [[Iris Epics]], E10 Destructive Operation Gates
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Close the tiered confirmation experience at conformance: all five destructive operations refuse without explicit confirmation, irreversible teardowns require typed-name confirmation on a TTY while dev-loop ops use a y/N prompt, `--dry-run` previews without writing, and `--force` cancels in-flight runs on the affected scope with the cancelled runs dead-lettered as stopped. This wires the E10.1 predicates into the real command surfaces.

## Depends on

Epics E03, E05, E06. Builds on [[E10.1 Gate and blocker predicates]].
