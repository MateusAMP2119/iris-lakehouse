# E05.10 Manual pipeline run

Epic: [[Iris Epics]], E05 Dispatcher, Lanes and Dead Letters
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Deliver manual iris pipeline run and the pipeline list surface. A manual run applies the depends_on gate exactly like a loop pass and consumes the upstream successes it ran against; an ineligible run exits 4 with a reason, and a dead-lettering manual run exits 5 with cause manual. Lane-member manual runs queue at the current run boundary preserving same-lane serialization, own-lane pipelines run immediately, and iris pipeline list shows active runs by default with --all expanding to every registered pipeline.

## Depends on

Epics E02, E03; builds on [[E05.5 Gate and consumption]].
