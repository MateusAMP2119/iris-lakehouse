# E02.1 Meta DDL and schema

Epic: [[Iris Epics]], E02 Engine Install, Daemon and Leadership
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Deliver the embedded bootstrap DDL as a pure, assertable foundation: the dedicated meta database with exactly its seventeen control tables, the partitioned data_journal in the data database, the exact foreign-key graph, and identity-column ordering keys. This locks the state model early (all state in Postgres, create-if-missing, no self-migration ledger) so every later task builds against a fixed schema.

Epic E01.
