# E08.2 Build and artifact storage

Epic: [[Iris Epics]], E08 Build, Artifacts and Modes
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Wire the explicit `iris pipeline build` path end to end: building compiles the source into one self-contained binary, records its content hash as an immutable row in the `artifacts` table (index entry only), and stores the bytes in the object store under that hash. Rebuilds insert new rows rather than mutating old ones, and declare apply never triggers a build. Actual PyInstaller/pkg conformance builds ride E13.

## Depends on

Epics E03, E05. Builds on [[E08.1 Recipe inference and matrix]].
