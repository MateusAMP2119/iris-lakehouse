# E06.2 Capture trigger emission

Epic: [[Iris Epics]], E06 Write Capture, Wipe and Promotion
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Emit the golden PL/pgSQL capture triggers and wire provisioning so every declared table gets one, unconditionally and with no opt-out. Triggers are statement-level with transition tables, issue one INSERT...SELECT per statement, add no columns or stamps to user tables, and both provisioning paths end by ensuring the journal and trigger coverage exist.

## Depends on

Epics E03, E05. Builds on [[E06.1 Journal DDL and partitioning]].
