# E09.6 Endpoint apply lifecycle

Epic: [[Iris Epics]], E09 Read API, Endpoints and PATs
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Take the compiled endpoint shapes live: iris endpoint apply prepare-verifies the derived SQL against the data database and persists atomically, applied endpoints serve requests on commit without a daemon restart, re-apply swaps shapes at a request boundary, and apply/remove run independently of declare apply and declare destroy. Integration work against a running daemon and real Postgres.

## Depends on

Epics E02, E03, E04; builds on [[E09.2 Endpoint compile and validation]] and [[E09.5 Route mux and auth]].
