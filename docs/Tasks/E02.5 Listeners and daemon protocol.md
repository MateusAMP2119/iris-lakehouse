# E02.5 Listeners and daemon protocol

Epic: [[Iris Epics]], E02 Engine Install, Daemon and Leadership
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Stand up the CLI-daemon protocol surface: the zero-config unix socket serving HTTP/JSON, the opt-in PAT-gated TCP listener with TLS when certs are given, and the .iris directory path defaults (socket, iris.toml, log files). Covers daemon lifecycle at the CLI edge: foreground by default with -d to detach, the fixed roster of daemonless commands, and fail-fast guidance when a daemon-touching command finds no reachable daemon.

## Depends on

Epic E01; builds on [[E02.4 Install and uninstall]].

| Contract | Behavior | Tier |

| `S02/daemonless-lifecycle-commands` | Only iris engine install, start, service install/uninstall, and uninstall on a stopped daemon are classified as runnable without a daemon. | unit |
