# E05.6 Failure propagation

Epic: [[Iris Epics]], E05 Dispatcher, Lanes and Dead Letters
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Deliver lazy failure propagation, pure unit work against the fake process runner. When a downstream's awaited run is dead-lettered, the dispatcher writes it a never-executed dead-lettered run with cause propagated, reason upstream_dead_lettered, failed_upstream naming the immediate upstream, and the poisoned run recorded in run_inputs. Propagation flows only along depends_on edges, computed lazily at consumption time, and is transitive edge by edge; composer-ordered neighbors and independent pipelines stay untouched.

## Depends on

Epics E02, E03; builds on [[E05.5 Gate and consumption]].
