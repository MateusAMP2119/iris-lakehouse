# E13.1 Golden workspace fixture

Epic: [[Iris Epics]], E13 Golden Sample and Acceptance
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Build the golden sample workspace itself: two tables, three single-script pipelines composed into one ingest lane, and one declared read endpoint. Pin its shape and its dependency-versus-composer-order split with unit tests, then prove that four single-file declare applies register the full sample graph with schema provisioning riding each apply. This fixture is shared by every later E13 task, so it starts as soon as E00 exists.

## Depends on

All other epics as they land (E13 is the spine that proves them); the fixture itself only needs E00.
