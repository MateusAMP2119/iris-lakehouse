# E08.4 Mode execution and retirement

Epic: [[Iris Epics]], E08 Build, Artifacts and Modes
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Close the mode matrix in the live engine: dev mode executes the pipeline source via its language runtime while built mode executes the content-addressed binary directly and ignores the run vector, with both modes receiving the injected scoped connection and full orchestration wiring. Finish with artifact retirement, where the dispatcher deletes post-prune artifact rows that are neither pipeline-newest nor referenced by a surviving run, along with their object-store bytes. This is the live-daemon closure of the epic.

## Depends on

Epics E03, E05. Builds on [[E08.2 Build and artifact storage]].
