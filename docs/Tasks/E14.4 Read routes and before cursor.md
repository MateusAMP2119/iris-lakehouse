# E14.4 Read routes and before cursor

Epic: [[Iris Epics]], E14 Graph Views and Triage Surface
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

The read routes an IDE-style renderer consumes. GET /workload serves the wiring payload, GET /runs?include=inputs embeds consumed upstream ids and replayed_from as plain row attributes, and the trace, gate, and impact routes serve the same walks the CLI prints with the verdict and status enums closed. The before= reverse keyset cursor lets log views page newest-first without any clock entering paging.

## Depends on

Epics E05, E07, E09. Builds before E13, the acceptance spine. Plus [[E14.1 Ref grammar and triage shows]] (the trace, gate, and impact routes serve the same walks the CLI prints) and [[E14.2 Workload wiring panel]] (the /workload route serves the same wiring payload the panel renders).
