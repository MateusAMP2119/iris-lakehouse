# E09.9 NDJSON streaming

Epic: [[Iris Epics]], E09 Read API, Endpoints and PATs
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Add NDJSON streaming to every collection route: Accept: application/x-ndjson streams one JSON row per line with no envelope, on the same routes with the same auth, and a dropped stream resumes by passing the last received row's key as the cursor. A small, natural seam layered on the finished route set and keyset paging.
