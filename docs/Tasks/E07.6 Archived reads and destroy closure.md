# E07.6 Archived reads and destroy closure

Epic: [[Iris Epics]], E07 Provenance, Journal Lifecycle and Object Store
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Close the epic against the live daemon: the ==provenance walk== spans the archive boundary (resident ranges from Postgres, archived ranges from object-store files) with equivalent results, missing objects fail by naming the hash, and an auditor can validate the full chain offline with only the archive files and the engine public key. Destroy writes archival summaries before deleting runs and preserves the engine, schemas/ tree, endpoints, and journal history, and the ==`iris data provenance`== CLI readout is verified against the shipped binary.

## Depends on

Epics E05, E06; builds on ==[[E07.1 Provenance walk]]== and [[E07.5 Object store and export]].
