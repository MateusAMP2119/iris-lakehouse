# E05.1 Exec seam

Epic: [[Iris Epics]], E05 Dispatcher, Lanes and Dead Letters
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Deliver subprocess execution as a clean seam: the engine starts a run by direct exec of the declared run command in its own process group with no shell, captures its output, and records the process-group id as the run handle and crash-recovery key. The engine never kills a run unilaterally; a run ends only by exiting on its own or by explicit cancellation, which kills the process group and dead-letters the run as stopped. This seam also yields the fake process runner that the later pure gate and propagation unit work builds against.
