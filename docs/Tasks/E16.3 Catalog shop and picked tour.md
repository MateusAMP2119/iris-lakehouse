# E16.3 Catalog shop and picked tour

Epic: [[Iris Epics]], E16 Install Ceremony and Pipeline Catalog
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

The shop inside the PIPELINE act. Painted browse list (number + name cyan, pitch dim),
prompt `Pick a pipeline (1-N, Enter=1):` on errOut; empty answer → entry 1; integer in
range → that entry; anything else (non-numeric, out-of-range, EOF, Ctrl-C) → clean
abort, exit 0 with resume hint (a typo never runs a command). Picked entry renders its
description + finale preview; the apply step's ordinary confirm is the confirmation —
no extra prompt. New `tourPick` seam (`func(question string, n int) (choice int, ans
promptAnswer, err error)`) joining the single shared stdin `bufio.Reader` (a type-ahead
line is never dropped between the pick and the next Y/n); cancellable via the existing
goroutine+context pattern. `--pipeline <id>` flag: explicit pick honored in every
rendering (interactive skips browse, detail still prints; `--yes`; plain guide; `--json`);
unknown id → usage error exit 2 naming available ids. `--yes` without `--pipeline` →
entry 1. Wrap-up, welcome, and the dead-letter lesson parameterized by the picked
entry. Plain guide gains the catalog block; `--json` envelope gains
`catalog{default, selected, entries[{id,name,pitch,showcase}]}` (additive); golden
re-pinned. Conformance leg: picked non-default tour per the contract row.

Spec text and contract rows landed in E16.1; this task claims only.
