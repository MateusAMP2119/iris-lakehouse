# E08.3 Promote gating

Epic: [[Iris Epics]], E08 Build, Artifacts and Modes
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Implement `iris pipeline promote` and its gates: promotion flips the pipeline's per-pipeline data_mode in meta from disposable to permanent, is refused whenever the pipeline is not in built state (so source-only pipelines can never hold permanent data), and repeats the cross-mode read warning while any upstream read dependency is still disposable. Gating checks are unit work; the meta flip is integration.

## Depends on

Epics E03, E05. Builds on [[E08.2 Build and artifact storage]].
