# E05.4 Lane model and walk

Epic: [[Iris Epics]], E05 Dispatcher, Lanes and Dead Letters
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Deliver the lanes table and walk construction: lanes are keyed by (lane, pipeline), reference pipelines by name so an order may name unregistered folders, and walks skip names with no registered pipeline while a registered pipeline named in no lane becomes its own anonymous lane. Composer order is pure sequencing: no data link, no failure propagation, no gating, and no implicit declaration-order tiebreak for unordered pipelines. This is mostly pure unit work on walk construction.

Epics E02, E03.
