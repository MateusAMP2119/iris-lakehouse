# E09.2 Endpoint compile and validation

Epic: [[Iris Epics]], E09 Read API, Endpoints and PATs
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Deliver the pure endpoint compile path: flat shape-only YAML files at endpoints/<name>.yaml, validation of the single-table projection, source resolution against schemas/, filter and sort rules, and deterministic derivation of exactly one parameterized SQL text. Also pins the endpoints and endpoint_filters store shape those compiled results persist into. Mostly unit tests against the compiler, no live daemon needed.

## Depends on

Epics E02, E03, E04.
