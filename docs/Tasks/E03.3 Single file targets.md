# E03.3 Single file targets

Epic: [[Iris Epics]], E03 Declarations, Schemas and Apply
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Enforce the single-file target semantics of iris declare apply and iris declare destroy: exactly one declaration file per invocation, folder paths resolved to their file, no workspace sweep, no --all flag, and a bare invocation exiting 2 as a usage error. This is pure argument and target validation at the CLI seam.

## Depends on

Epic E01; builds on [[E03.1 Declaration parsing and discovery]].
