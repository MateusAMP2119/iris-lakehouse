# E03.7 Migration ledger sync

Epic: [[Iris Epics]], E03 Declarations, Schemas and Apply
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Deliver the additive-only migration machinery end to end: diffing table.yaml against the ledger head, appending immutable engine-written migration files, running the corresponding ADD COLUMN ALTERs, recording applied heads in the migrations meta table, auto-fixing additive drift (missing columns, missing capture triggers), and a dry-run mode that touches nothing. The recording pg fake with golden DDL files carries most of this integration weight.

## Depends on

Epics E01, E02; builds on [[E03.5 Type mapping and DDL]] and [[E03.6 Drift classification]].
