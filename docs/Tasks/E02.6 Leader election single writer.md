# E02.6 Leader election single writer

Epic: [[Iris Epics]], E02 Engine Install, Daemon and Leadership
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Implement leader election on the session advisory lock (held on a session-pinned connection) and the single-writer meta path: exactly one leader dispatches, all meta writes flow through the one dispatcher goroutine, the CLI never touches meta directly, standbys reject mutations with guidance, and readers use plain MVCC connections with no busy-retry so meta stays readable while the daemon runs.

## Depends on

Epic E01; builds on [[E02.5 Listeners and daemon protocol]].
