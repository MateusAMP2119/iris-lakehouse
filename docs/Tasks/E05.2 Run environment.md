# E05.2 Run environment

Epic: [[Iris Epics]], E05 Dispatcher, Lanes and Dead Letters
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Resolve what a run's subprocess starts with: working directory set to the pipeline folder and an environment built from the inherited env, declared env map entries (literals or ${HOST_VAR} interpolations), env_file contents re-read fresh by the leader at each dispatch, and the injected scoped DB connection, with explicit env winning over env_file on key collisions. No runtime parameters exist anywhere, so the yaml declaration fully determines every run.

## Depends on

Epics E02, E03; builds on [[E05.1 Exec seam]].
