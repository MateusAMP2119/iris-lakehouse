# E05.8 Dead letter drain

Epic: [[Iris Epics]], E05 Dispatcher, Lanes and Dead Letters
Spec: [[Iris Specification Inventory]] (source of truth; on conflict the spec wins)

## Goal

Deliver drain and the explicit-scope rule for the deadletter commands. Drain is pure discard: it re-runs nothing, alters no downstream, makes the run prunable, and a drained run can never be replayed afterward. Both replay and drain refuse a bare invocation, requiring an explicit <run>, --pipeline, or --all scope (exit 2), and drain touches only the entries within the given scope.

## Depends on

Epics E02, E03; builds on [[E05.7 Dead letter replay]].
